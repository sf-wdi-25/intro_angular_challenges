Challenge #1: Activating Angular

  1. Copy paste the follwing script tag into the <head> tag:
    <script src="https://cdnjs.cloudflare.com/ajax/libs/angular.js/1.4.9/angular.js"></script>
  2. Add the attribute "ng-app" to the html tag, like so:
    <html ng-app>
  3. You should see "Angular is working!" rendered below:


{{ "Angular isn't working!".replace("isn't", "is") }}

You can think of logic inside double curly braces
as being like the <%= "clownhats" + "sprinkles" %>
from erb templates in rails!


Exercises

Fill in the blanks:
  1. 1 + 1 = {{ 1 + 1 }}
  2. "hello" + "world" = {{ "hello" + "world" }}
  3. ["a","b","c"] = {{ ["a","b","c"] }}
  4. ["a","b","c"][2] = {{ ["a","b","c"][2] }}
  5. true = {{ true }}
  6. nil = {{ nil }}
  7. null = {{ null }}
